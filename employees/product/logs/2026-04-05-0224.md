## Instagramカルーセル投稿

---

### 1枚目：タイトル（強いフック）

```
がんばってるのに
疲れが取れない人の正体

あなたは消耗している
じゃなくて──

「整え方を知らないだけ」
```

---

### 2枚目：共感（読者の痛みを言語化）

```
こんな経験、ない？

朝起きてもスッキリしない
休んでも疲れが残る
なんかずっとモヤモヤしてる

それ、サボってるからじゃない。

ちゃんとやってるからこそ
消耗してしまってる。

"がんばること"に
慣れすぎているだけ。
```

---

### 3枚目：原因分析

```
なぜ整わないのか？

原因は"エネルギーの使い方"にある。

消耗するがんばりの特徴：
・やらなきゃと焦って動く
・休むことに罪悪感がある
・感情を無視して続ける

これが三週間続くと
ストレスホルモンが慢性的に出て
体に症状があらわれる。

「整える」のは
サボりじゃなくて、回復。
```

---

### 4枚目：解決策

```
「整える」ってどういうこと？

整えるとは、
自分の感覚を無視しないこと。

「なんか嫌だな」を飲み込まない。
「疲れた」を「まだいける」で上書きしない。

一日一回、自分に聞いてみて。
「今、どんな気持ち？」

それだけで、
三日後の自分がちがってくる。
```

---

### 5枚目：具体的手順①

```
整えるための朝のルーティン

① 起きてすぐスマホを見ない
　（最初の五分は自分の感覚に集中）

② 三回、ゆっくり深呼吸する
　（副交感神経が優位になる）

③ 「今日もいい日になる」を五回唱える
　（潜在意識への小さな投げかけ）

たった十分で
一日のエネルギーが変わる。
```

---

### 6枚目：具体的手順②

```
夜の"リセット習慣"

① 寝る前に今日の「よかったこと」を一つ書く
　（ポジティブ回路を育てる）

② 使い心地のいいアイテムで感覚を整える
　（香り・感触が副交感神経に働く）

③ 「今日もよかった」と声に出す
　（一日を肯定して終わらせる）

この三つを五日間続けると
朝のズシッとした感覚が軽くなる。
```

---

### 7枚目：まとめ

```
今日のまとめ

疲れが取れない人は
がんばりすぎてるんじゃなくて
「整え方を知らない」だけ。

✔ 感情を無視しない
✔ 朝の五分を自分のために使う
✔ 夜は「よかったこと」で締める

これを続けた三週間後、
あなたの朝は変わってる。
```
