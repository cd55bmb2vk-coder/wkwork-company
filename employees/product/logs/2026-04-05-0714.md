# 大学生×AI副業の現実：256万稼いで翌月ゼロになった僕が、今また動き出せた理由

---

## 1. あなたに正直に話す

副業で稼いでいる話なんて、ネットに溢れている。

「月100万達成！」「3ヶ月でフォロワー1万人！」

そういう投稿を見るたびに思う。「失敗した話は？」と。

僕は大学生で個人事業をしている。物販と制作で累計1,000万円以上を売った。でも去年、256万円を稼いだ翌月に売上がゼロになった。

その話をする。

成功談だけ語るのはズルいと思っている。失敗も全部さらけ出すことが、読んでくれるあなたへの誠実さだと思っている。

そしてこのnoteは、そんな僕がAIを使って今また前に進んでいる、その記録だ。

---

## 2. なぜ256万稼いだ翌月にゼロになったのか

あの月のことを今でも覚えている。

数字が動いて、口座の残高が増えていくのを見ていた。「やった。これが事業だ」と思った。浮かれていた。

でも翌月、数字は動かなかった。1件も入らなかった。

最初は「タイミングが悪いだけ」と思っていた。でも2週間経っても、3週間経っても、ゼロのままだった。

原因を分析した。出てきた答えはシンプルだった。

**「なぜ売れたか、分析していなかった」**

前月に売れた理由は「たまたまSNSでバズったから」だった。バズりは一時的なもので、再現性がなかった。仕組みがなかった。

つまり、稼いだのではなく「拾った」だけだった。

この違いが分かったとき、初めて事業の本質を理解した気がした。

---

## 3. 「仕組みのない成功」は砂の城だ

副業で稼ぐ人の多くが同じ失敗をしている。

最初の1回が売れる。「これだ！」と思う。でもそれが「なぜ売れたか」を分析しないまま同じことを繰り返す。

売れている間はいい。でも売れなくなった瞬間に何もできない。

砂の城と同じだ。綺麗に見えても、波が来たら崩れる。

じゃあ何が必要か。

**「再現性のある動線」だ。**

- どんな人に
- どんな価値を
- どんな方法で届けるか

この3つが設計されていれば、1ヶ月売れなくても「次は何を直せばいいか」が分かる。

256万ゼロの経験は、僕にとって最高の授業料だった。

---

## 4. 失敗から見えた、本当に必要なもの

売上ゼロの月、正直しんどかった。

「自分には向いていないのかも」と思った日もある。「やめようか」と考えた夜もある。

でも、動き続けた。なぜか。

**「失敗している今が一番価値になる」**と分かったから。

成功した人の話は世の中に溢れている。でも「今まさに失敗中」の話をリアルタイムで語れる人は少ない。

これが差別化になると気づいた。

そして気づいたのが「仕組み化」の必要性だ。

毎回ゼロから考えるのではなく、AI × 自分の経験 を組み合わせて「再現できる収益の仕組み」を作ること。

それが今の僕の取り組みだ。

---

## 5. AIで副業を「仕組み化」する具体的な手順

AIを使えば副業の仕組みを作れる。でも「AIに丸投げ」は機能しない。

大事なのは「AIを骨組みに、自分の経験を肉にする」こと。

### ステップ1：ペルソナを決める

まず「誰に届けるか」を明確にする。

ChatGPTに聞くとやりやすい。

```
「副業を始めたいと思っている大学生で、
スキルはないけど稼ぎたい人が抱えている
悩みを5つ挙げてください」
```

AIが出してきた悩みの中から、自分が実際に経験したものを選ぶ。経験のないものは書かない。

### ステップ2：コンテンツの骨組みを作る

ペルソナの悩みが決まったら、構成をAIに作らせる。

```
「『大学生が副業でゼロから稼ぐ方法』というテーマで
noteの構成を5つの見出しで作ってください。
読者は副業未経験の大学生です」
```

出てきた構成を見て、「自分が語れるか」を確認する。語れない見出しは削る。

### ステップ3：自分の体験を肉付けする

構成ができたら、各見出しに「自分の実体験」を入れる。

AIが生成した文章そのままを使わない。必ず自分の言葉に変換する。

ここが最重要だ。AIが作れないのは「あなたの失敗談」と「あなたの感情」だ。

### ステップ4：投稿して反応を見る

書いたら無料公開で反応を測る。

いいね・保存・コメントの数を見て「どの部分が刺さったか」を分析する。

反応がよかった部分を深掘りして、次の記事のテーマにする。これがサイクルだ。

### ステップ5：有料コンテンツに転換する

反応がよかった無料記事の「続き」「詳細版」「テンプレ付き」を有料にする。

「無料で価値を渡しすぎ」という心配は不要だ。無料で価値を受け取った人が、もっと欲しくなって有料を買う。

---

## 6. 大学生だからこそできる最強のポジショニング

「大学生で副業」というポジションは思った以上に強い。

なぜか。

社会人の副業発信者は多い。でも「今まさに失敗中の大学生」は少ない。

読者は「成功した人の話」より「自分に近い人の話」に共感する。

フォロワー10人の今の僕が語ることには、フォロワー10万人の人には語れない価値がある。

**「成長過程をリアルタイムで見せること」が最大の差別化だ。**

256万→ゼロという失敗体験も、AIで復活中という今の状況も、全部コンテンツになる。

「何も持っていないから発信できない」と思っている人へ。

違う。今の状態が一番の武器だ。

このnoteを読んで「やってみたい」と思ったなら、今日中に動いてほしい。

```

知りたいと思っている疑問を10個出してください」
```

出てきた10個の中から、自分が一番詳しく語れるものを選ぶ。

**ステップ2：構成を作って1本書く（今週中）**

完璧な記事は後でいい。まず1本書くことが大事だ。1,500〜2,000字でいい。

**ステップ3：無料公開して反応を測る（来週まで）**

note、X、どちらでもいい。出してみなければ何も分からない。

反応がゼロでも落ち込まない。最初はゼロが普通だ。僕も今ゼロから始めている。

## 8. まとめ：失敗は最強の資産になる

256万稼いで翌月ゼロになった経験は、当時は最悪だと思っていた。

でも今は、あの失敗がなければ「仕組み」の大切さを理解できなかったと思っている。

失敗した経験がある人は、すでに「成功している人が語れない話」を持っている。

それを価値に変える方法が、AI × コンテンツ副業だ。

# 【有料部分】僕が実際に使っているAIプロンプト集と月次収益化ロードマップ

## なぜ有料にしたか

「無料でいいじゃないか」と思う人もいるかもしれない。

でも正直に言う。これは限定公開にしたかった。

本当に動く気がある人だけに届けたい。無料にすると、読み飛ばされる可能性が高い。

500円払って読んでくれた人の方が、確実に実行してくれると信じているから有料にした。

今すぐ手に入れてほしい。このロードマップに沿って動けば、最短で初収益を出せる。

---

## 実際に使っているプロンプト集

### プロンプト1：ペルソナ深掘り

```
あなたは副業コンサルタントです。
以下の条件の読者が抱えている悩みを、
感情も含めて詳しく5つ教えてください。

条件：
・[ターゲット属性を入力]
・副業を始めたいが何から始めればいいか分からない
・失敗が怖くて一歩踏み出せない
```

### プロンプト2：フック生成

```
以下のテーマでXに投稿するための
「フック（1文目）」を10パターン作ってください。

テーマ：[テーマを入力]

条件：
・共感を呼ぶか、好奇心を刺激するもの
・「自分のことだ」と思わせる表現
・数字か逆説を使う
```

### プロンプト3：note構成生成

```
以下の読者に向けたnoteの構成を
5〜7つの見出しで作ってください。

読者：[ターゲットを入力]
テーマ：[テーマを入力]
自分の体験：[簡単に入力]

条件：
・冒頭3行で読者の痛みに触れる
・問題提起→原因→解決策→手順→まとめの流れ
```

### プロンプト4：リライト・口語化

```
以下の文章を読みやすくリライトしてください。

条件：
・一文60文字以内で切る
・漢字率30%以下
・「〜です。〜ます。」の連続禁止
・断定調と混ぜる
・数字を具体的に（「たくさん」→「30件」）

[文章を貼り付け]
```

### プロンプト5：タイトル生成

```
以下の記事内容に合った
noteのタイトルを5パターン作ってください。

条件：
・検索されるキーワードを含める
・読者が「自分のことだ」と思う表現
・数字型・暴露型・逆説型それぞれ含める

内容概要：[内容を入力]
```

## 月次収益化ロードマップ

### 1ヶ月目：土台を作る

**目標**: noteを5本書く・Xで毎日投稿・フォロワー50人を目指す

やること：
- ChatGPTでテーマ10個出す → 一番得意な1つを選ぶ
- 無料note 3本 + 有料note 2本（単価300〜500円）
- X投稿：1日1〜2本（失敗談・学びシェア）
- 他の同ジャンルアカウントに積極的にリプ

---

### 2ヶ月目：反応を分析する

**目標**: noteのPV 500を超える・有料記事の初売上を出す

やること：
- 1ヶ月目で反応がよかった投稿の深掘り記事を書く
- 有料noteの価格を300円から800円に上げる
- X投稿でCTA（「保存しないと損します」）を意識的に使う
- note・Xのアクセス解析を毎週確認する

---

### 3ヶ月目：仕組みを固める

**目標**: 月収1万円の初達成・フォロワー200人

やること：
- 毎週1本の有料note（単価800〜1,500円）をルーティン化
- 人気記事をマガジン化（月額マガジン 500〜1,000円）
- Xで「保存・リポストされる投稿」の型を3つ確立する
- 他のクリエイターとコラボ投稿でお互いの読者に紹介し合う

---

### 6ヶ月目の目標状態

- noteフォロワー：300〜500人
- 月収：3〜5万円
- Xフォロワー：500〜1,000人
- 有料コンテンツ：10〜15本

この数字は僕の目標でもある。一緒に達成しよう。
