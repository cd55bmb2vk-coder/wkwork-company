## カルーセル構成（8枚）

---

### 1枚目：タイトル（強いフック）

```
なんかずっとしんどい人へ。

```

### 2枚目：共感

```
毎日ちゃんと動いてる。

でも夜になると、なんか空っぽ。
理由もないのに涙が出そうになる。

悪いことしてないのに
なんでこんなに疲れてるんだろう、って
思ったことない？

それ、あなただけじゃない。
```

---

### 3枚目：気づき①

```
【気づき①】
頑張りすぎは「自分を後回し」のサイン

人のために動ける人ほど
自分のエネルギーが枯れていく。

まず自分を満たさないと
誰にも何も与えられない。

電池が切れたままで
走り続けてるようなものだよ。
```

---

### 4枚目：気づき②

```
【気づき②】
比べるのは「自分を見失ってる」サイン

SNSで誰かの成功を見て
「なんで自分は…」って思う瞬間。

それって相手を羨んでるんじゃなくて
自分の本音が叫んでる状態。

比べる必要なんてない。
あなたはあなたのペースでいい。
```

---

### 5枚目：気づき③

```
【気づき③】
「なんか違う」という感覚を大事にして

理由が言えなくても
「なんかいやだな」って思うなら
それは正解のサインかもしれない。

体と心は正直で
頭より先に答えを知ってる。

直感は、使えば使うほど鋭くなる。
```

---

### 6枚目：気づき④

```
【気づき④】
整える習慣は「毎日15分」から始まる

特別なことをしなくていい。
夜、スマホを置いて
好きな香りをたいて
ただ呼吸するだけ。

三日続けると体感が変わる。
一週間続けると朝が変わる。

小さな習慣が、静かに人生を変える。
```

---

### 7枚目：まとめ

```
今日のまとめ

① 頑張りすぎは「後回し」のサイン
② 比べるのは「迷子」のサイン
③ 直感は正直なメッセージ

しんどい日がある、それは普通のこと。
でも「整える方法」を知ってれば
ちゃんと戻ってこられる。

保存しておくと、しんどい時に使える。
```
